Project Overview

SYNRAXIS is a symbolic entropy modeling framework built on the deep structural analysis of the Collatz system. Its aim is to uncover symbolic patterns, invariant motifs, entropy shifts, and reconvergence pathways in chaotic systems. This prototype later extended to forecasting celestial orbits and energy states using symbolic terrain models.
